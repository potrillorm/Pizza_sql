Pizza Shop Database Project 🍕
This repository contains a database project I created for my ITM 220 class. I designed a relational database for a fictional pizza shop and populated it with mock data to practice and learn how to write SQL queries.

What's in this repository:
Entity-Relationship Diagram (ERD): A visual blueprint of the database schema and table relationships.

Database Setup Scripts: A copy of the SQL files containing the CREATE TABLE and INSERT statements needed to recreate the database and the dummy data I have been working with.

Final Semester Project: A collection of SQL scripts that demonstrate my ability to write complex, advanced queries.

Streamlit Frontend Application: A mini frontend app built with Python and Streamlit. This program connects directly to my local MySQL server, providing a user-friendly interface to insert new data and access existing records without needing to write SQL manually.

Technologies Used
Database: MySQL

Language: SQL, Python

Frontend: Streamlit (pizzas.py)

Libraries: mysql-connector-python, streamlit
